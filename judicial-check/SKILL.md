# 裁判書查詢技能

## 觸發時機

當用戶說「檢查 [姓名]」或「查詢 [姓名]」時觸發。
可選擇性地指定地區：北部、中部、南部

例如：
- 「檢查林志穎」
- 「檢查林志穎 北部」
- 「查詢王小明 中部」

## 地區對應法院

| 地區 | 法院 |
|------|------|
| 北部 | 臺北、新北、桃園、士林、基隆、宜蘭、新竹地方法院 |
| 中部 | 臺中、彰化、南投、雲林、苗栗地方法院 |
| 南部 | 高雄、臺南、屏東、嘉義、臺東、花蓮、橋頭地方法院 |

## 工作流程

### 使用 browser 工具 (OpenClaw 內建)

1. **導航到搜尋頁面**：
   ```
   browser action: navigate
   targetUrl: https://judgment.judicial.gov.tw/FJUD/default.aspx
   ```

2. **輸入姓名**：
   - 找到輸入框 (ref: e63)
   - 點擊並輸入姓名
   ```
   browser action: act
   request: {"kind": "type", "ref": "e63", "text": "姓名"}
   ```

3. **點擊搜尋按鈕** (ref: e65)

4. **分析結果與回報**：
   - 取得 snapshot
   - 解析案件清單
   - 解析案件詳情，提取：
     - 關係人角色（被告、被害人、證人等）
     - 事件摘要

### 輸出格式

#### 案件基本資訊
- 裁判字號、日期、法院、案由

#### 關係人角色（重要！）
- **被告**：刑事案件中被起訴的人
- **告訴人/被害人**：案件的受害者
- **證人**：在案件中作證的人
- **共犯**：與被告共同犯罪的人
- **聲請人**：民事案件中提出聲請的人
- **相對人/債務人**：民事案件中的對造
- **原告/上訴人**

#### 事件摘要
- 簡述案件發生經過（時間、地點、原因）

#### 危險分類
- 🔴 危險：毒品、洗錢、傷害、恐嚇、槍砲、強制、殺人、盜竊
- 🟡 警示：博弈、違反法規
- 🟢 一般：民事案件（債務、裁定等）

#### 輸出範例

```
## 搜尋結果：許俊泰 - 中部

### 總筆數：2 筆

---

### 🔴 危險紀錄：0 筆

### 📌 結論
- 中部無危險紀錄
- 需留意：同名可能性

### 地區篩選方式

如有指定地區，使用「更多條件查詢」：
1. 點擊「更多條件查詢」連結
2. 選擇特定法院（根據地區）
3. 輸入姓名搜尋

或搜尋全部後，按法院分類統計結果。

## 危險案件關鍵字

搜尋結果中可能表示危險的關鍵字：
- 傷害
- 恐嚇
- 槍砲
- 毒品
- 毒品危害防制條例
- 強制
- 公共危險
- 殺人
- 搶奪
- 竊盜
- 侵占
- 偽造文書
- 洗錢
- 組織犯罪
- 妨害風化
- 少年事件

## 注意事項

- 搜尋結果可能包含同名不同人，請標註此限制
- 判決書為公開資料，如實呈現結果即可
- 不需要點進去看詳細內容，除非用戶特別要求
